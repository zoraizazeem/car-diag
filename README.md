# This is a mission statement page


Why should we create a better car diagnostic system?


Simply put the exisitng system doesn't work.


Dealerships are the only places with the authorative car diagnostic solutions, that are tailor made for the make (and indeed all models).

But which consumer likes taking there car directly to the dealer to get it fixed, considering the price discrepancy with independant repair shops?

This sets the case for the push for independant third party repair softwares (the creation of which are protected by the right to repair laws).

State of third party diaganostic tools:

consist of packaged tools (with expensive Ipads offshoots with limited quality control and exorbinant prices)


There is some ground covered already with many third party independant softwares filling the gap of diagnostic tools.

The most selling is the VCDS software, that uses it's own dedicated HEX-V2 hard ware and old software that has limited upgrade aside from base new integrations.

All other software (bimmergeeks, autodiag etc.) either only cover some or parts of the diagnostic parts either duew fialures of the hardware (J234 passthru) or some lack of clarity in development.

The USP will be the AI that will help in digesting the codes and aid in diagnostics.
